ValueSet: Newborn Congenital Anomalies

Official URL: http://hl7.org/fhir/us/bfdr/ValueSet/ValueSet-newborn-congenital-anomalies Version: 2.0.0-Preview3
Active as of 2024-07-30 Computable Name: NewbornCongenitalAnomaliesVS

Copyright/Legal: This value set includes content from SNOMED CT, which is copyright © 2002+ International Health Terminology Standards Development Organisation (IHTSDO), and distributed by agreement between IHTSDO and HL7. Implementer use of SNOMED CT is not covered by this agreement

The valueset contains codes to represent newborn congenital anomalies. Includes ‘Other’ value for general utility, although #OTH is not allowed for birth submissions to NCHS.
