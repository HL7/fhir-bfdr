Capability Statements

CapabilityStatement - Birth and Fetal Death

This section describes the expected capabilities of a BFDR Document producer actor who is responsible for producing clinical documents and a BFDR Document consumer who receives and consumes the clinical documents.

Bundles

Bundle - Document Birth Report

This Bundle profile represents a Birth Document Bundle. It contains any one of the Birth Report Compositions.
Bundle - Document Coded Cause of Fetal Death

This bundle profile contains instances of the resources comprising cause of fetal death coded content.
Bundle - Document Demographic Coded Content

This bundle profile contains instances of the resources comprising demographic (race and ethnicity) coded content. This bundle is information-content equivalent to the traditional NCHS MRE/NRE format.
Bundle - Document Fetal Death Report

This Bundle profile represents a Fetal Death Document Bundle. It contains any one of the Fetal Death Compositions.

Compositions EHR to VRO

Composition - Provider Fetal Death Report

This Composition profile contains constraints to address the use case describing the need for fetal death information to be recorded and communicated to the jurisdictional Vital Records Office.
Composition - Provider Live Birth Report

This Composition profile defines constraints to address the use case in which information for live birth information is recorded and communicated to the jurisdictional Vital Records Office.

Compositions VRO to NCHS

Composition - Jurisdiction Fetal Death Report

This Composition profile contains information of a fetal death and the creation of a jurisdictional file to be recorded and communicated to the national statistics agency.
Composition - Jurisdiction Live Birth Report

This Composition profile contains information of a live birth and the issuance of a Birth Certificate to be recorded and communicated to the national statistics agency.

Compositions NCHS to VRO

Composition - Coded Cause of Fetal Death

This profile communicates coded cause of fetal death information to appropriate jurisdictional Vital Records Offices.
Composition - Coded Race and Ethnicity

This Composition profile communicates coded race and ethnicity information to the appropriate jurisdictional Vital Records Office, along with the submitted input race and ethnicity information.

Profiles Newborn/Fetus

Observation - APGAR Score

APGAR score post-birth. Migrated from VRCL.
Observation - Birth Weight

The weight of the infant/fetus at birth/delivery. Migrated from VRCL.
Observation - Fetal Presentation at Birth/Delivery

This Observation profile represents the fetal presentation (orientation within the mother’s womb) of a fetus prior to delivery/birth. Only the presentations in the preferred binding can be reported to NCHS. The full set of abnormal presentations is included to support more general use.
Observation - Gestational Age at Delivery

The obstetric estimate of the infant’s gestation in completed weeks based on the birth/delivery attendant’s final estimate of gestation which should be determined by all perinatal factors and assessments such as ultrasound, but not the neonatal exam. For submission to NCHS, values in days will be divided by 7 without remainder, and values in weeks will be rounded down to an integer.
Observation - Infant Breastfed at Discharge

This profile represents whether or not the infant was being breastfed at discharge.
Observation - Infant Living

An indication of whether the infant is living at the time the report is being submitted. Migrated from VRCL.
Observation - Mother Delivery Weight

The weight of the mother at the time of birth/delivery. Migrated from VRCL.
Observation - Number of Fetal Deaths This Delivery

The number of fetal deaths in this delivery. Migrated from VRCL.
Observation - Number of Live Births This Delivery

The number of live births in this delivery. Migrated from VRCL.

Mother Prenatal Profiles

Coverage - Principal Payer for Delivery

This Coverage profile represents the principal payer for this delivery.
Observation - Cigarette Smoking Before and During Pregnancy

This profile represents the average number of cigarettes smoked per day for each time period indicated by the value selected for code.
Observation - Date of First Prenatal Care Visit

This profile represents the date of the first prenatal care visit for this pregnancy. A prenatal visit is where a physician or other health care professional examined and/or counseled the pregnant woman for the pregnancy. If this profile is not present it indicates that the mother did not receive prenatal care at any time during the pregnancy.
Observation - Date of Last Live Birth

This profile represents the date of the mother’s last live born infant (do not include this birth).
Observation - Date of Last Other Pregnancy Outcome

This profile represents the date of the mother’s last other pregnancy outcome (includes pregnancy losses of any gestational age, e.g., spontaneous or induced losses or ectopic pregnancies) (do not include this birth).
Observation - Last Menstrual Period

This profile represents the date of the last menstrual period of the patient. The first day of last menstrual period should be captured. The effectiveTime captures the observation date. Migrated from VRCL.
Observation - Mother Height

The height of the mother. Migrated from VRCL.
Observation - Mother Prepregnancy Weight

The weight of the mother before becoming pregnant. Migrated from VRCL.
Observation - Mother Recieved WIC Food

This profile represents whether or not the mother made use of the Women, Infants and Children (WIC) nutritional program during the pregnancy.
Observation - Number of Births Now Dead

The total number of previous live-born infants for the mother now dead. Migrated from VRCL.
Observation - Number of Births Now Living

The total number of previous live-born infants for the mother now living. Migrated from VRCL.
Observation - Number of Other Pregnancy Outcomes

The total number of other pregnancy outcomes for the mother that did not result in a live birth (included pregnancy losses of any gestational age, e.g., spontaneous or induced losses or ectopic pregnancies). Migrated from VRCL.
Observation - Number of Prenatal Visits

The total number of prenatal visits for the mother. The dates of the first and last prenatal visit are indicated by the effective date range. If the mother received no prenatal care, the value should indicate 0 visits. Migrated from VRCL.

This profile represents whether or not the mother was married at conception, at the time of birth, or at any time between conception and giving birth.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Units of Gestational Age Vital

This valueset contains the set of codes for specifying the units used when recording gestational age (days, weeks.).

Reporting to NCHS is always in units of weeks, but other users of the associated [ObservationGestationalAgeAtDelivery] profile may want to record days.
